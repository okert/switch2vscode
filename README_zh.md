### 从预构建包安装
1. 从 `build` 文件夹下载预构建的ZIP包
2. 打开您的JetBrains IDE
3. 进入 `设置` → `插件` → `⚙️` → `从磁盘安装插件...`
4. 选择下载的ZIP文件
5. 点击 `确定` 安装
6. 重启IDE以激活插件
